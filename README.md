SimpleETL
=========
ETL tools are often very general and (for XML) very slow. So lets try if we can do it better ourselves ;)

Requirements
============
* Java SDK 1.8
* Gradle
* GDAL/OGR

GDAL/OGR (Windows)
==================
* Download from http://www.gisinternals.com/sdk/
* add [INSTALLDIR]/bin and [INSTALLDIR]/bin/gdal/java to PATH
